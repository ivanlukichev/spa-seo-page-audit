# Privacy Policy

`SPA: SEO Page Audit` analyzes the currently open page locally in the browser.

## What Data Is Processed

- Current page URL
- Page title and meta tags
- Headings
- Visible text statistics
- Links and anchor names
- Images and alt attributes
- Structured data blocks
- Open Graph and Twitter tags

## How Data Is Used

- Data is used only to generate on-page SEO analysis inside the extension.
- Analysis runs locally in the browser on the user's machine.
- No external SEO APIs or third-party analysis services are used.

## What Data Is Not Collected

- No account data is collected.
- No page content is sent to external servers by the extension.
- No analytics, tracking scripts, or remote telemetry are included for page analysis.

## Temporary Storage

- Recent analysis results may be stored temporarily by tab ID inside the extension background layer and session storage.
- This temporary storage is used only to support popup mode, full page mode, and re-analysis workflows.
- Stored analysis is cleared when the related browser tab is closed or when the page URL changes.

## Permissions

The extension uses browser permissions only to:

- Access the active tab
- Read page data from supported `http` and `https` pages
- Store short-lived session data for analysis state

## Restricted Pages

Browser-protected pages such as internal browser URLs, extension pages, or store pages cannot be analyzed due to browser security restrictions.
