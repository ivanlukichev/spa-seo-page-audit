# SPA: SEO Page Audit

Browser extension for fast local on-page SEO analysis.

This repository contains browser-ready extension builds only.

- [extension/chrome](./extension/chrome)

## Notes

- Analysis runs locally in the browser
- No external SEO APIs are used
- This is not an official Google score
