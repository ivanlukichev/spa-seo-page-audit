# SPA: SEO Page Audit

![SPA: SEO Page Audit banner](./extension/readme-banner.png)

SPA: SEO Page Audit is a browser extension for fast on-page SEO checks of the current page.

It helps you quickly spot common SEO issues on site pages directly in the browser: missing titles and descriptions, weak heading structure, empty anchors, missing image alt text, missing schema, social meta gaps, and other page-level problems that affect crawlability, snippets, and on-page quality.

## Why It Exists

When you work on a website, it is easy to miss small on-page SEO errors that quietly hurt visibility:

- missing or weak `title` and `meta description`
- no `H1` or broken heading structure
- missing `canonical` or `robots` hints
- image links with poor anchor naming
- images without `alt`
- missing schema or social tags

SPA: SEO Page Audit makes these issues visible in a few clicks, without external SEO APIs and without sending page content to third-party services.

## Why It Matters

Good on-page SEO helps pages:

- communicate their topic more clearly to search engines
- generate better snippets in search results and social previews
- avoid avoidable technical mistakes
- improve internal QA before release or publication

## Who It Is For

- SEO specialists doing quick page-level audits
- developers checking technical SEO details during implementation
- site owners and editors reviewing pages before publishing
- agencies and freelancers who need a lightweight browser-side audit tool

## What The Extension Checks

- page title and meta description
- canonical, robots, viewport, lang and other basic technical signals
- headings structure
- internal and external links
- accessible anchor names
- images and alt coverage
- schema and social metadata
- page-level score, issues and quick wins
