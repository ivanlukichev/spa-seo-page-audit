# Extension Builds

This folder contains browser-specific builds of `SPA: SEO Page Audit`.

Available folders:

- `chrome/` for Chrome
- `edge/` for Microsoft Edge
- `opera/` for Opera
- `firefox/` for Firefox
- `safari/` for Safari via Xcode and Safari Web Extension packaging

Each browser folder includes:

- The full unpacked extension build
- A browser-specific `manifest.json`
- A `README.txt` with local loading steps
- A `STORE_DESCRIPTION.md` file with store listing copy

Safari differs slightly:

- `safari/` contains an Xcode project generated from the web extension
- The Safari app and extension are packaged through Xcode / App Store flow
